# 两个图床速度比拼

```javascript
刚上线博客, 突发奇想
既然选择用到图床, 为何不搞一场比赛呢. 在一个页面中展示两个图床的加载速度.
```

下面先介绍一下, 我选择的两款图床.

## 路过图床
    - 国内图床
    - 最大单张支持 10 MB
    - 全球CDN加速, 支持外链, 原图保存
    - https://imgse.com/

## imgur
    - 国外老牌图床
    - 最大单张支持: 应该是 10 Mb 以上
    - 全球CDN加速, 支持外链, 原图保存
    - https://imgur.com/
